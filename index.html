Mathematical Background

The Sierpiński problem (1964) asks: Are there infinitely many positive integers that cannot be expressed in any of the four forms:

  • 6ab + a + b = (3a + 1)(2b + 1) - 1
  • 6ab + a - b = (3a + 1)(2b - 1) - 1
  • 6ab - a + b = (3a - 1)(2b + 1) - 1
  • 6ab - a - b = (3a - 1)(2b - 1) - 1

Status: This problem remains UNSOLVED - one of mathematics' enduring mysteries.

First uncovered numbers: 2, 3, 7, 8, 12, 18, 27, 28, 30, 32, 33, 35, 42, 43, 45, 47, 48, 50...
