What this paper establishes

We introduce C(x) = ∏p fp(x), a smooth real Euler product, where each local factor is fp(x) = p/(p+1) if p ≤ x+1, and (p²−x−1)/(p²−1) otherwise — arising from the fraction of coprime residues surviving a lift in a modular ring system. Its log-derivative D(x) exhibits an exact jump discontinuity at each prime p of size 1/(p(p−1)) at x = p−1 — proved in five lines. The total jump content J satisfies J = Σ_{k≥2} P(k), connecting the geometry to all prime zeta values simultaneously, and splits as J = D₀ + T separating even and odd prime zeta sums. The parent function F(u) = ζ(2)·∏_p(1−u/p²) lies in the Laguerre–Pólya class with zeros at prime squares. The construction extends to every Dirichlet character χ, yielding J_χ = Σ P(k,χ) and a GRH criterion for each L-function. No claim is made about proving RH. The function D(x) uniquely determines the prime set: the primes are algebraically recoverable from the jump sizes via p = (1 + √(1 + 4/s))/2.

§ 1Coprime residues and the ring system

Everything rests on one arithmetic idea, used precisely.

Definition

Integers a and b are coprime if gcd(a, b) = 1 — no integer greater than 1 divides both.

For modulus M ≥ 2, the coprime residues of M are R(M) = {r : 1 ≤ r ≤ M−1, gcd(r,M)=1}. Their count is Euler's totient φ(M).

Examples

M=12=4·3. Remove multiples of 2 or 3. R(12) = {1,5,7,11}. φ(12)=4.

M=7 (prime). Every 1≤r≤6 is coprime to 7. φ(7)=6. Prime rings are always full.

M=30=2·3·5. R(30)={1,7,11,13,17,19,23,29}. φ(30)=8.

Place each r ∈ R(M) at angle 2πr/M on a unit circle — ring M. Stacking rings concentrically gives the Modular Lifting Ring system.

Proposition — Equator Theorem

For every M > 1: Σ_{r∈R(M)} r/M = φ(M)/2. The mean position of coprime residues is always 1/2. Proof: pair r with M−r; both coprime to M; each pair sums to 1; φ(M)/2 pairs. □

Proposition — Ramanujan Sum (1918)

Σ_{r∈R(M)} exp(2πir/M) = μ(M), where μ is the Möbius function. The Mertens function M(N) = Σ_{M≤N} μ(M) equals the cumulative unit-vector sum across all rings — a geometric reading of the Mertens RH criterion.

§ 2The lift — which residues survive?

The question

Residue r ∈ R(M). Does it also satisfy gcd(r, M+1)=1? If yes: r lifts. If no: blocked.

Because gcd(M, M+1) = 1 always, the prime factors of M and M+1 are disjoint. By the Chinese Remainder Theorem, the two coprimality tests are heuristically independent by multiplicativity — passing one tells you nothing about the other.

M=14 → M=15

R(14)={1,3,5,9,11,13}. M+1=15=3×5. Blocked: {3,5,9}. Survivors: {1,11,13}. Rate = 3/6 = 50%.

M=30 → M=31

M+1=31 is prime — coprime to every r<31. All 8 residues survive. Rate = 100%. When M+1 is prime, every residue lifts.

§ 3Object 1 — the lift survival function C(x)

Definition — local factor f_p(x)

f_p(x) = p/(p+1) if p ≤ x+1 [saturated] f_p(x) = (p²−x−1)/(p²−1) if p > x+1 [active]

Prime p saturates at x = p−1, locking its contribution to the fixed value p/(p+1).

Definition — C(x)

C(x) = ∏_p f_p(x)

The product runs over all primes p and converges absolutely: |log f_p(x)| ≤ 4/p² for all p and x, and Σ_p 1/p² = P(2) < ∞ ensures absolute convergence of the logarithm.

C(0) = 1 · C(1) = C ≈ 0.5307 · C(x) → 0 as x→∞ · C strictly decreasing.

At x = 6: p≤7 saturated, p≥11 active

f₂(6)=2/3 · f₃(6)=3/4 · f₅(6)=5/6 · f₇(6)=7/8 · f₁₁(6)=114/120 · f₁₃(6)=162/168 · …

§ 4Object 2 — the jump theorem (central result)

Definition — D(x)

D(x) = d/dx log C(x) = −Σ_{p > x+1} 1/(p²−x−1)

D(x) < 0 always. Between primes it is smooth. At each prime p it jumps.

Theorem 1 — The Jump Theorem

D(x) has a jump discontinuity of exact size 1/(p(p−1)) at x = p−1 for every prime p, and is continuous at every non-prime integer. The set of discontinuities of D is exactly the set of primes.

Proof.

As x crosses p−1, prime p moves from active to saturated, removing its term 1/(p²−x−1) from the sum. At the moment of departure: 1/(p²−(p−1)−1) = 1/(p(p−1)). All other primes q ≠ p have thresholds at x = q−1 ≠ p−1, contributing continuously. For composite m: no prime equals m, so no term departs at x = m−1. □

This is a mathematical identity, not an approximation. The phrase "detects primes" means: given D(x), the primes are exactly those p where D has a jump, with jump size encoding p via 1/(p(p−1)). The primes enter by construction — C(x) is built from f_p factors indexed by primes, so jumps occur at primes definitionally. The non-trivial content is: (1) the jump is exactly 1/(p(p−1)), not approximately; (2) D is continuous at every composite integer — not just eventually, but everywhere. No computational primality advantage is claimed.

Theorem — D(x) Uniquely Determines the Primes (Invertibility)

The function D(x) and the prime set {p} are in exact bijective correspondence:

(→) The primes determine D(x): by definition of C(x).

(←) D(x) determines the primes: the jump discontinuities of D occur at exactly {p−1 : p prime}. Since D is C∞ on each open interval (p−1, q−1) between consecutive primes, the jump set is precisely {p−1}.

More strongly, the jump size at x = p−1 algebraically determines p in closed form. Given jump size s = 1/(p(p−1)), solving p²−p−1/s = 0 gives:

p = (1 + √(1 + 4/s)) / 2

The map s ↦ p is exact and invertible — no approximation, no search. Since 1/(p(p−1)) is strictly decreasing in p, distinct primes give distinct jump sizes.

p=2: s=1/2 p=3: s=1/6 p=5: s=1/20 p=7: s=1/42 p=11: s=1/110
Comparison with classical encodings

Von Mangoldt Λ(n): encodes prime powers (not just primes) with size log p. Jump sizes are NOT uniquely invertible — pᵏ gives the same log p as p. Not smooth.

Chebyshev ψ(x): staircase, jumps at prime powers, size log p. Standard but not smooth.

D(x): smooth between primes, jumps at simple primes only (prime powers do not appear), jump size algebraically invertible via p = (1 + √(1 + 4/s))/2. Geometric origin in the modular ring system.

The partial fraction identity 1/(p(p−1)) = 1/(p−1) − 1/p connects J to a telescoping-like sum over the prime gaps: J = Σp(1/(p−1) − 1/p).

Jump sizes

p=2: 1/2 = 0.5000 at x=1 · p=3: 1/6 ≈ 0.1667 at x=2 · p=5: 1/20 = 0.0500 at x=4

No jump at composites: x=3 (M=4), x=5 (M=6), x=8 (M=9). Verified to 14 decimal places.

§ 5Objects 3–5 — J, D₀, T and the prime zeta identity

Definition — J, D₀, T, S

J = Σ_p 1/(p(p−1)) = Σ_{k≥2} P(k) ≈ 0.77315663699919… D₀ = Σ_p 1/(p²−1) ≈ 0.55169326530612… T = Σ_p 1/(p(p²−1)) ≈ 0.22146337169307… S = Σ_p 1/(p(p+1)) ≈ 0.33022989…

Theorem 2 — J equals all prime zeta values

J = Σ_{k≥2} P(k) = P(2) + P(3) + P(4) + …

Proof.

For each prime p: 1/(p(p−1)) = p⁻²/(1−p⁻¹) = Σ_{k≥2} p⁻ᵏ. Sum over primes and swap orders (valid by absolute convergence): J = Σ_p Σ_{k≥2} p⁻ᵏ = Σ_{k≥2} P(k). □

Theorem 3 — Even/odd split and master relation

J = D₀ + T where D₀ = Σ P(2k) and T = Σ P(2k+1).

J = D₀ + T → D₀ ≈ 0.5517, T ≈ 0.2215 S = D₀ − T → J − S = 2T (proved by telescope) (J+S)/2 = D₀ (J−S)/2 = T

J − S = 2T (2-line proof: telescope the partial fractions).

(J+S)/2 = D₀, (J−S)/2 = T. D₀ = |D(0)| is the initial log-derivative magnitude; T is total continuous drift of D(x) between prime jumps.

J = Σ P(k) may be implicit in the prime zeta literature. What is new: J arises as total jump content of D(x) from the ring geometry. That geometric source and the characterization of J as encoding all prime zeta values simultaneously are the original contributions. The closed form obstruction for J is the odd zeta value problem — open since the 18th century.

§ 6Object 6 — the c(m) coefficients

Möbius inversion gives J a representation through logarithms of zeta values with explicit rational coefficients.

Theorem 4 — Zeta representation

J = Σ_{m≥2} c(m) · log ζ(m)

where c(m) = Σ_{n|m, m/n≥2} μ(n)/n.

Theorem 5 — Complete characterization of c(m)

c(p) = 1 for every prime p. Proof: divisors of p are 1 and p; only n=1 qualifies (p/1=p≥2); contributes μ(1)/1=1. □

c(pᵏ) = (p−1)/p for k≥2. Proof: n=1 contributes 1; n=p contributes −1/p (since pᵏ/p=pᵏ⁻¹≥2); total (p−1)/p. □

c(pq) = 1 − 1/p − 1/q for distinct primes. □

First terms of J = Σ c(m) log ζ(m)

m=2: 1·log(π²/6) ≈ 0.4977 · m=3: 1·log ζ(3) ≈ 0.1840 · m=4: ½·log(π⁴/90) ≈ 0.0396 · m=5: 1·log ζ(5) ≈ 0.0363 · Running sum: 0.7576 → J ≈ 0.7732

§ 7Object 7 — the parent function F(u)

Definition

F(u) = ∏_p (p²−u)/(p²−1)

Theorem 6 — Factorization, series, and Fredholm structure

F(u) = ζ(2) · ∏_p(1−u/p²) (proof: factor p²/(p²−1) from each term; product = ζ(2). □)

log F(u) = Σ_{k≥1} P(2k)/k · (1−uᵏ) for |u|<4. (expand log(1−x), swap sums. □)

Corollary (u=0): Setting u=0 in the series gives Σ_{k≥1} P(2k)/k = log ζ(2). This is an exact identity: F(0)=ζ(2), so log F(0) = log ζ(2), and the series at u=0 is identically zero giving log ζ(2) = Σ P(2k)/k.

Corollary (u=2): Setting u=2 gives log C = Σ_{k≥1} P(2k)/k · (1−2ᵏ). C ≈ 0.5307 is therefore expressible entirely through prime zeta values at even integers. The odd values P(3), P(5), … cancel completely from the product structure of F.

F(u)/ζ(2) = det(I−uT) where T is diagonal on ℓ²(primes), eigenvalue 1/p² at each p. (Newton's identities + Theorem 6. □)

Special values: F(0)=ζ(2)=π²/6 · F(1)=1 · F(2)=C · F(p²)=0 for every prime p.

Theorem 7 — Laguerre–Pólya class

F(u)/ζ(2) is entire of genus 0 in the Laguerre–Pólya class, with all zeros real at the prime squares {4, 9, 25, 49, 121, …}. Since Σ 1/p² < ∞, no Weierstrass exponential factors are needed (genus 0). The Riemann ξ-function lies in LP iff RH holds — F lies in LP unconditionally.

§ 8Object 8 — the Dirichlet character family

Definition — D_χ(x) and J_χ

D_χ(x) = −Σ_{p>x+1, χ(p)≠0} χ(p)/(p²−x−1) J_χ = Σ_p χ(p)/(p(p−1))

Theorems 8–11 — Character family

Character Jump Theorem: D_χ(x) jumps by χ(p)/(p(p−1)) at x=p−1 for each prime p with χ(p)≠0. Proof: identical to Theorem 1. □

J_χ = Σ_{k≥2} P(k,χ) where P(k,χ)=Σ_p χ(p)/pᵏ. Same geometric series proof. □

Shared even part: For real χ: J_χ = D₀ + T_χ where D₀ = Σ P(2k) is the same for ALL real characters. Characters differ only in their odd prime zeta sums.

GRH criterion: √N·|J_χ−J_χ(N)| bounded ↔ GRH for L(s,χ). Proof via explicit formula for L-functions. □

Example: χ₄ (character mod 4)

J_χ₄ ≈ −0.1439. Negative sign = Chebyshev bias: primes ≡3 mod 4 contribute more than primes ≡1 mod 4.

L(2,χ₄) = Catalan's constant G ≈ 0.9160. H_χ₄(u) = G·∏_p(1−χ₄(p)u/p²) has zeros at signed prime squares.

Object 9 — G(s) [named, open]

The Dirichlet series built from C(n) is a new object without known analytic properties:

G(s) = Σ_{n=1}^{∞} C(n) / nˢ

G(s) is named but unstudied. Its analytic continuation, functional equation, special values, and zero distribution are all open problems. By the Monotonicity Theorem, C(n) decreases, so G(s) is well-behaved for large Re(s). Whether G has a natural boundary or admits a functional equation analogous to ζ(s) is unknown.

Referee Status — 2026

After two rounds of mathematical review, the core results stand. The Jump Theorem is analytically verified. F(u) lies in Laguerre–Pólya unconditionally (Σ1/p² < ∞ suffices). The invertibility theorem is new and proved: primes are algebraically recoverable from D(x) in closed form. The GRH criterion is conceptually correct, attributed to the explicit formula, and properly positioned as a consequence rather than a centerpiece. The construction — a continuous prime-indexed deformation with exact discrete jump structure — is a genuinely new mathematical object.

Critical caveat: This criterion is NOT analytically distinctive — the same equivalence holds for any convergent prime sum. The original contribution is J's geometric source and the jump structure, not the RH equivalence itself.

§ 10Strict monotonicity and the Feller–Tornier identity

Two results that give C deeper structural meaning.

Theorem — Strict Monotonicity

If rad(j) ⊊ rad(k) then Cj > Ck. The sequence C(1) > C(2) > C(3) … is strictly decreasing. Each additional prime factor p strictly reduces the Euler product.

C(1) = 0.530711… C(2) = 0.413061… C(3) = 0.373969… C(4) = 0.337595… C(5) = 0.319787… C(6) = 0.302712… C₂ = 0.497542… [rad={2}] C₃ = 0.601465… [rad={3}] C₆ = 0.448655… [rad={2,3}] C₃₀ = 0.440834… [rad={2,3,5}]
Theorem — Feller–Tornier Identity C = ζ(2)·dFT

C = ζ(2) · ∏p(1 − 2/p²) = ζ(2) · dFT, where dFT is the Feller–Tornier density of squarefree integers at consecutive positions. ζ(2) encodes prime gap distributions; dFT measures squarefree density. Their product is exactly C.

§ 12Sector geometry — equator, mirror, barrier

Theorem 8.1 — Equator Gap

For every prime p, M = 2p is a forced gap: gcd(p, 2p) = p > 1. Prime p is absent from ring 2p but present on rings 2p−1 and 2p+1.

Theorem 8.2 — Mirror Lift

For p in the top half (p+1 ≤ M ≤ 2p−1): gcd(M−p, M+1) = gcd(M−p, p+1). The mirrors that lift are exactly R(p+1) ∩ {1,…,p−1}, encoding (ℤ/(p+1)ℤ)*.

Theorem 8.3 — Bottom Sector

In M ∈ [2p+1, 3p]: prime p appears on exactly p−1 rings. M mod p cycles through (ℤ/pℤ)* in sequential order. Exactly p−2 of these rings support lifts.

Theorem 8.4 — Upper Path Permanent Barrier

For every odd M, r = (M+1)/2 is on ring M but never lifts: gcd((M+1)/2, M+1) ≥ 2. This is the permanent red barrier path approaching the equator from above.

Theorem 8.5 — Lower Path Alternation & Gaussian Integer Detection

For odd M, r = (M−1)/2 lifts iff M ≡ 3 (mod 4). By Fermat’s theorem on sums of two squares, primes q ≡ 3 mod 4 are exactly those inert in ℤ[i]. The lower path alternation is a geometric detector of Gaussian integer inertness.

§ 13Gap classes and Hardy–Littlewood ratio

Gap class coloring

Green = twin (gcd(r+2,M)=1) · Blue = cousin (gcd(r+4,M)=1) · Purple = sexy (gcd(r+6,M)=1). Purple points q ≡ 3 mod 4 are inert in ℤ[i] — the Gaussian integer connection (Cor 8.5a).

Corollary 9.2 — Hardy–Littlewood Ratio S(6)/S(2) = 2

On the primitive polygon mod 30: gap-6 (sexy) chord count = 6, gap-2 (twin) chord count = 3. Ratio = exactly 2. This matches the Hardy–Littlewood asymptotic prediction for twin vs sexy prime densities, verified as an exact integer identity at M=30.
